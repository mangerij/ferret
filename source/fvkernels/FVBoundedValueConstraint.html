FVBoundedValueConstraint

This object implements the residuals that enforce the constraint

or

using a Lagrange multiplier approach. E.g. this object enforces the constraint that the value of in the domain has to be above or below a certain value.

warningwarning

The contribution to the diagonal of the system of this kernel is null, which introduces a saddle point. Make sure to use a NONZERO shift in your preconditioner.

Example input syntax

In this example, the value of the variable v is forced to be positive using a FVBoundedValueConstraint. This accelerates the convergence to the solution, as the initial condition is negative and the solution is everywhere positive.

[Variables]
  [v]
    type = MooseVariableFVReal
  []
  [lambda]
    type = MooseVariableScalar
  []
[]

[FVKernels]
  [diff]
    type = FVDiffusion
    variable = v
    coeff = coeff
  []
  [average]
    type = FVPointValueConstraint
    variable = v
    phi0 = 13
    lambda = lambda
    point = '0.3 0 0'
  []
[]
(../moose/test/tests/fvkernels/constraints/point_value.i)

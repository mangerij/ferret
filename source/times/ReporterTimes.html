ReporterTimes

Import times from one or more reporters, for example other Times

In order to retrieve a Reporter with the ReporterTimes, it must be a vector of floating point numbers (std::vector<Real>).

The times will be extracted from all the reporter vectors and sorted.

Example File Syntax

In this example, the ReporterTimes is combining the times from two other Times, using the fact that Times objects are Reporters in the back-end.

[Times]
  [input]
    type = InputTimes
    times = '0.2 0.4 0.9'
    outputs = none
  []
  [input_2]
    type = InputTimes
    times = '0.2 0.5 0.6'
    outputs = none
  []
  [reporter]
    type = ReporterTimes
    reporters = 'input/times input_2/times'
  []
[]
(../moose/test/tests/times/reporter_times.i)

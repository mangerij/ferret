HeatFluxTensor

buildconstruction:Undocumented Class

The HeatFluxTensor has not been documented. The content listed below should be used as a starting point for documenting the class, which includes the typical automatic documentation associated with a MooseObject; however, what is contained is ultimately determined by what is necessary to make the documentation clear for users.

heat flux generated

Overview

Computes the component of the heat flux vector (dependent on tensor quantities),

The tensor components , and correspond to the thermal conductivity, Seebeck, and electrical conductivity respectively which require information of the material crystallographic directions.

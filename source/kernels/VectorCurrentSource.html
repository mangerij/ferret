VectorCurrentSource

Kernel to calculate the current source term in the Helmholtz wave equation.

Example Input File Syntax

[current_real]
  type = VectorCurrentSource
  variable = u_real
  function_coefficient = -1.0
  source_real = source_real
  source_imag = source_imag

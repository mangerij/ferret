SLKKS multi-phase model kernel to enforce . The non-linear variable of this kernel is a phase's sublattice concentration

For a sub lattice KKS (SLKKS) model with phases, the phase concentration constraint equation is where indexes a component, a phase, and a sublattice in the given phase. is the fraction of sublattice sites in phase . The version of this class for a two phase model with a single switching function is SLKKSPhaseConcentration.

    Controllable:No

    Controllable:No

    Description:The name of the variable that this residual object operates on

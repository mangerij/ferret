SimpleSplitCHWRes

Gradient energy for split Cahn-Hilliard equation with constant Mobility for a coupled order parameter variable.

commentnote

This kernel is part of a set of simplified phase field kernels that have been created for educational purposes. These kernels use simplified formulations with limitations that prohibit spatially varying mobilities and gradient energies. The source code of these kernels is kept simple without using advanced C++ features.

This kernel implements the weak form for the a reverse split Cahn-Hilliard equation. For a more feature complete version see the SplitCHWRes kernel, which computes a full Jacobian even for variable dependent mobilities .

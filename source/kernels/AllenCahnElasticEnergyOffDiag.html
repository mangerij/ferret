AllenCahnElasticEnergyOffDiag

This kernel calculates off-diagonal Jacobian of elastic energy in AllenCahn with respect to displacements

Calculates the off-diagonal term

is the elastic energy, is the strain. The first term on the right is a material property called 'd2Fdcdstrain'.

Example Input File Syntax

[./off_disp]
  type = AllenCahnElasticEnergyOffDiag
  variable = c
  displacements = 'disp_x disp_y'
  mob_name = L
[../]
(../moose/modules/combined/test/tests/phase_field_fracture/crack2d_aniso.i)

SideIntegralFunctorPostprocessor

Computes a surface integral of the specified functor, using the single-sided face argument, which usually means that the functor will be evaluated from a single side of the surface, not interpolated between both sides.

The SideIntegralFunctorPostprocessor is also an intermediate base class that should be derived from for any calculation involving the integral of a functor quantity over a side.

The "functor_argument" parameter lets the user select between face integration and element side quadrature point integration. Face integration is more appropriate for finite volume variables, and quadrature point integration is more appropriate for finite element variables.

Input Parameters

  • boundaryThe list of boundary IDs from the mesh where this object applies

    C++ Type:std::vector<BoundaryName>

    Controllable:No

    Description:The list of boundary IDs from the mesh where this object applies

  • functorThe name of the functor that this postprocessor integrates

    C++ Type:MooseFunctorName

    Controllable:No

    Description:The name of the functor that this postprocessor integrates

    Default:TIMESTEP_END

    C++ Type:ExecFlagEnum

    Options:NONE, INITIAL, LINEAR, NONLINEAR, TIMESTEP_END, TIMESTEP_BEGIN, MULTIAPP_FIXED_POINT_END, MULTIAPP_FIXED_POINT_BEGIN, FINAL, CUSTOM, ALWAYS, TRANSFER

    Controllable:No

    Description:The list of flag(s) indicating when this object should be executed, the available options include NONE, INITIAL, LINEAR, NONLINEAR, TIMESTEP_END, TIMESTEP_BEGIN, MULTIAPP_FIXED_POINT_END, MULTIAPP_FIXED_POINT_BEGIN, FINAL, CUSTOM, ALWAYS.

    C++ Type:MooseEnum

    Description:Location to evaluate the functors at

    Default:1

    C++ Type:MooseFunctorName

    Controllable:No

    Description:Factor multiplying the integrand

    Default:False

ConstantPostprocessor

Postprocessor that holds a constant value

Overview

The ConstantPostprocessor is used to hold a constant value. This postprocessor may be useful as a general controllable postprocessor for the Controls system.

Example Input File Syntax

In this example, the value of the ConstantPostprocessor recv is controlled by a RealFunctionControl.

[Controls]
  [func_control]
    type = RealFunctionControl
    parameter = 'Postprocessors/recv/value'
    function = 'val'
    execute_on = 'timestep_begin'
  []
[]

[Functions]
  [val]
    type = ParsedFunction
    expression = '1 + 2*t'
  []
[]

[Postprocessors]
  [recv]
    type = ConstantPostprocessor
  []
[]
(../moose/test/tests/postprocessors/constant/receiver.i)

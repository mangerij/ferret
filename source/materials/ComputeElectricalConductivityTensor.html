ComputeElectricalConductivityTensor

Store an electric conductivity tensor

Overview

Computes (and stores) the electrical conductivity tensor at each quadrature point in the finite element mesh. The orientation of the tensor can be rotated via,

for some arbitrary rotation matrices using the internal RotationTensor operation in MOOSE utils. The rotation operator accepts Euler angles in the standard Bunge sequence (). Note that is generally the nomenclature for electrical conductivty, not to be confused with the linear elastic stress.

ComputeLagrangianStrainAxisymmetricCylindrical

Compute strain in 2D axisymmetric RZ coordinates.

Overview

This strain calculator functions the same way as ComputeLagrangianStrain. The only difference is that the strain calculated by this material is defined in the 2D axisymmetric cylindrical coordinates. See this page for the definition of the gradient operator in the 2D axisymmetric cylindrical coordinates.

NodalVariableStatistics

Nodal reporter to get statistics for a coupled variable. This can be transfered to other apps.

Overview

NodalVariableStatistics produces the following statistics for a variable: Maximum, Minium, Average, Total Nodes. The "base_name" can be used to prepend a name to each reporter.

Example Input File Syntax

[Reporters]
  [nodal_stats]
    type = NodalVariableStatistics
    coupled_var = u
    base_name = diffusion
  []
[]
(../moose/test/tests/reporters/nodal_reporter/nodal_stats.i)
